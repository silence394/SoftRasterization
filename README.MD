这是一个简易的可编程的软件渲染器。重点实现了软件渲染相关的部分，没有对应用层做过多的封装。
接口模仿了DX11，清晰明了，可以编写shader做想要的效果。

## 技术点
- 创建窗口，渲染设备并显示颜色
- 矩阵的位移、旋转、缩放变换
- 摄像机矩阵变换，摄像机控制
- 投影矩阵变换
- 拣选模式，支持全拣选，正面拣选，背面拣选
- 近裁剪面裁剪
- 视口变换
- 视口裁剪
- 线框模式，Bresenham画线算法
- 扫描线光栅化，左上填充规则
- 透视矫正
- 深度测试
- 自定义shader
- 材质
- Blinn-Phong 光照模型
